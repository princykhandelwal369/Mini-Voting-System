# 🗳️ Mini Voting System in C

A simple console-based voting system built using C.  
Users can vote for candidates, view live results, percentages, and see the winner.

---

## ✨ Features

- Vote for candidates
- Live vote counting
- Vote percentage calculation
- Winner detection
- Menu-driven interface
- Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

- C Programming
- GCC Compiler
